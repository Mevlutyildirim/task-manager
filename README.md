# TASK-MANAGER 

task-manager a project management app which you can efficiently track your project status. 

## Engineering Stack 

the app has been constructed with following techs.

* typeScript
* react
* styled-components
* node.js
* express
* mysql (with typeorm)
* redis
* elasticSearch
